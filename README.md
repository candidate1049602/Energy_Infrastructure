# Energy_Infrastructure
Jupyter notebooks for both parts of Energy Infrastructure assignment - MSc Energy Systems 2020-2021

PART A - Balearic Islands Network Assessment
    Jupyter Notebook uploaded, along with 3 excel files used to run it, for reference:
    Island Demand.xlsx, Hourly Averaged Installed Generation.xlsx, and Solar Generation.xlsx
    Information extracted from renewables.ninja or REE website.
   
PART B - Jamaica New Housing development
  Jupyter notebook uploaded, along with PV_Values.xlsx, used to add static generation to the network in the modeling.
  Data extracted from renewables.ninja
